# Blue Archive Characters Database

Welcome to the **Blue Archive Characters Database**! This project is a modern web application designed to help fans explore a comprehensive collection of data from the popular mobile game, *Blue Archive*.

## About the Project

Developed using React and modern CSS styling, the core focus of this project is to provide a seamless and visually appealing experience for searching and viewing detailed character information. 

Please note that all character assets, models, and illustrations showcased here are the property of their respective creators. The data and media featured were sourced from the community-driven [Blue Archive Wiki](https://bluearchive.wiki/wiki/Characters) and are utilized strictly for educational and portfolio purposes.

## UPDATE: AI-Assisted UI/UX Enhancements ✨

I've used AI to improve the UI/UX of the application to do the following:
- **Mica Effects:** A beautiful, frosted-glass translucent styling across the application components.
- **Glow Accents:** Dynamic, theme-based drop-shadow glows for interactive elements, resulting in a premium, modern feel.
- **Responsive Layouts:** Improved component layouts using Grid and Flexbox for a seamless experience across varying screen sizes.

# Base de Datos de Personajes - Blue Archive (Español)

¡Bienvenido a la **Base de Datos de Personajes de Blue Archive**! Este proyecto es una aplicación web moderna diseñada para ayudar a los fanáticos a explorar una colección exhaustiva de datos del popular juego para móviles, *Blue Archive*.

## Sobre el Proyecto

Desarrollado utilizando React y estilos CSS modernos, el enfoque principal de este proyecto es proporcionar una experiencia fluida y visualmente atractiva para buscar y visualizar información detallada de los personajes.

Ten en cuenta que todos los recursos, modelos e ilustraciones de los personajes mostrados aquí son propiedad de sus respectivos creadores. Los datos y archivos multimedia incluidos fueron obtenidos de la [Wiki de Blue Archive](https://bluearchive.wiki/wiki/Characters) impulsada por la comunidad, y se utilizan estrictamente con fines educativos y para demostración en portafolio.

## Actualización: Mejoras de UI/UX Asistidas por IA ✨

He usado IA para mejorar la UI/UX de la aplicación para hacer lo siguiente  :
- **Efectos "Mica":** Un hermoso estilo translúcido tipo vidrio esmerilado en los principales componentes de la plataforma.
- **Acentos Luminosos (Glow):** Resplandores dinámicos basados en sombras para elementos interactivos, logrando una presentación súper moderna y premium.
- **Diseños y Layouts:** Mejoras en la estructura de los componentes utilizando Grid y Flexbox para obtener una navegación mucho más fluida.
